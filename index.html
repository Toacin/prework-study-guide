Prework Study Guide

HTML

  • The "head" element contains information about the webpage
  • "meta" contains broswer data
  • "title" contains tab title data
  • The body element represents the visible content shown to the user
  • "header" element adds header section to "body" element
  • "main" element denotes the main portion of the "body"
  • The "H1/H2/H3..." element adds headers
  • The "p" element adds a paragraph
  • The "ul/ol/li" elements add lists. (ul-unordered | ol-ordered | li-list elements)
  • The "footer" element adds footnotes.
  • The "img" element adds image, requires "src" and "alt"

CSS

  • A margin indicates how much space outside of the element box
  • A padding indicates how much space around content inside of element box
  • * applies CSS elements to all HTML elements
  • background color applies a base color to an element
  • color changes font color
  • class allows us to assign HTML elements to a class, and use said class as a selector for CSS elements

Git

  • git status: lists current git branch
  • git checkout feature/xyz: change branch (add "-b" before branch directory to create branch)
  • git add -A: stage all files for commit
  • git commit -m "xyz": commit all changes
  • git pull origin main: pull in the most up-to-date code
  • git push origin feature/xyz: push the committed changes to feature branch in repository
